## About The Project

[![KukerTUI in action][product-screenshot]](https://example.com)

KukerTUI merupakan aplikasi TUI (Textual User Interface) untuk mengisi logbook KKN di portal Universitas Riau. KukerTUI hadir dikarenakan kelalaian dalam mengisi logbook sehingga logbook harian yang harus diisi menumpuk. 

Tumpukan logbook harian tersebut jika diisi secara langsung di browser, maka akan memakan banyak waktu karena hanya bisa dicopy-paste satu-persatu. Dengan KukerTUI maka mahasiswa, dapat langsung mengcopy-paste semua kolom yang harus diisi dengan sekali copy-paste saja.

KukerTUI dibuat menggunakan Python dengan bantuan selenium.
